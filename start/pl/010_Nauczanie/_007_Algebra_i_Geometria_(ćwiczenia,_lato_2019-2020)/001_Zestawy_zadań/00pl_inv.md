<center>
[\[źródło: prof. dr hab. Jacek Golak\]](http://users.uj.edu.pl/~golak/)

Dowieść metodą indukcji zupełnej:

* $\sum_{k = 1}^{n} (2 k - 1) = n^{2}$,
* $\sum_{k = 1}^{n} a_{1} q^{k - 1}= a_{1} \frac{1 - q^{n}}{1 - q}$, $q \ne 1$,
* $\sum_{k = 1}^{n} k^{2} = \frac{1}{6} n (n + 1) (2 n + 1)$.
